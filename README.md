## Description
NestjsでREST,graphql,grpcでそれぞれAPIを作ってみたリポジトリ

|                |        |                                            | 
| -------------- | ------ | ------------------------------------------ | 
| 開発環境       | Docker | ![](https://skillicons.dev/icons?i=docker) | 
| フレームワーク | NestJs | ![](https://skillicons.dev/icons?i=nest)   | 
| DB             | MySQL  | ![](https://skillicons.dev/icons?i=mysql)  | 
| ORM            | Prisma | ![](https://skillicons.dev/icons?i=prisma) | 

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
